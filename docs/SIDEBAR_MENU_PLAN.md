## 2. 상세 메뉴 기능 설명

### 2.1 📊 대시보드 (Dashboard)

#### 2.1.1 📈 통합 현황판
**기능:** 전체 시스템 상태 및 주요 지표 모니터링
- **메타데이터 현황:** 등록된 테이블, 컬럼, 모델 수
- **표준화 현황:** 표준화율, 미표준화 대상 수
- **데이터 흐름:** 분석된 흐름 수, 최근 업데이트
- **AI 활용:** 질의 수, 성공률, 사용자 활동
- **시스템 상태:** 연계 시스템 상태, 성능 지표

#### 2.1.2 📋 작업 현황  
**기능:** 사용자별 작업 및 승인 대기 현황
- **내 작업:** 진행 중인 표준화, 모델링 작업
- **승인 대기:** 표준, 모델, 형상 승인 대기 목록
- **최근 활동:** 최근 수정, 생성, 승인된 항목들
- **알림:** 중요 공지, 시스템 점검, 업데이트 정보

### 2.2 🗂️ 데이터 표준 관리 (Data Standards)

#### 2.2.1 📝 단어/용어 관리
**기능:** 데이터 표준의 기본 단위인 단어와 용어 관리
- **단어 등록/수정/삭제**
- **용어 정의 및 관리**
- **동의어/유사어 관리**
- **금칙어 관리**
- **승인 워크플로우**

#### 2.2.2 🏷️ 도메인 관리
**기능:** 데이터 타입 및 제약 조건 관리
- **논리 도메인 정의**
- **물리 데이터 타입 매핑**
- **제약 조건 설정**
- **확장성 고려 설계**
- **컴플라이언스 대응**

#### 2.2.3 📊 코드 관리
**기능:** 공통코드 및 개별코드 통합 관리
- **통합코드 관리** (상위/하위, 분류유형)
- **개별코드 관리** (부여기관, 코드체계)
- **코드 통합 및 정비**
- **유사도 분석**
- **사용현황 모니터링**

#### 2.2.4 📊 표준화 현황
**기능:** 표준화 진행 상황 및 결과 모니터링
- **표준화 진행률**
- **미표준화 대상 목록**
- **표준화 결과 분석**
- **품질 지표**

#### 2.2.5 📊 표준 준수율  
**기능:** 시스템별 표준 준수 현황 모니터링
- **시스템별 준수율**
- **테이블별 준수율**
- **미준수 항목 상세**
- **개선 계획**

### 2.3 🏗️ 데이터 모델링 (Data Modeling)

#### 2.3.1 📐 ERD 관리
**기능:** 데이터 모델 설계 및 관리
- **ERD 작성/편집**
- **모델 버전 관리**
- **주제영역별 관리**
- **모델 표준 검증**
- **AI 기반 추천**

#### 2.3.2 🔧 모델 신청/승인
**기능:** 모델 변경에 대한 신청 및 승인 프로세스
- **신청서 작성**
- **승인 워크플로우**
- **영향도 분석**
- **승인 이력 관리**

#### 2.3.3 📋 형상 관리
**기능:** 개발/테스트/운영 환경 간 형상 관리
- **환경별 형상 비교**
- **DDL 스크립트 생성**
- **배포 관리**
- **형상 이력 추적**

#### 2.3.4 🔍 모델 검증
**기능:** ERD와 물리DB 간 일치성 검증
- **Gap 분석**
- **품질 검증**
- **표준 준수 검사**
- **자동 검증 규칙**

#### 2.3.5 📊 모델 현황
**기능:** 전체 모델 현황 및 통계
- **모델 통계**
- **사용 현황**
- **최근 변경사항**
- **품질 지표**
